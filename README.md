# JavaScript LocalStorage Mini Apps

This repository contains multiple JavaScript mini projects demonstrating the use of the **LocalStorage API** and **real-time form validation**.

## Projects Included

### 1. Dark Mode Toggle
- Toggle between light and dark theme.
- Theme preference is saved using **localStorage**.
- The selected theme remains even after refreshing the page.

### 2. Sticky Notes App
- Create and delete notes.
- Notes are stored in **localStorage**.
- Notes remain saved even after page reload.

### 3. Registration Form with Real-Time Validation
Features:
- Inline validation for **Name, Email, and Phone**
- **Password Strength Validation** including:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
  - Minimum length requirement
- **Show / Hide password toggle**
- Stores form submissions in **localStorage** as a `submissions` array.

## Technologies Used
- HTML
- CSS
- JavaScript
- LocalStorage API
